How you can get involved

Some of the ways in which you can get involved:

  • OPEN UP DATA – share and publish data with an open licence.

  • INNOVATE WITH DATA – use opportunity data to build interesting tools and experiences.

  • LEARN ABOUT DATA – take part in ODI-certified training to ensure organisations understand what open data is and how publishing data can benefit them.

  • CREATE DATA STANDARDS – collaborate to create common standards for consistent, reliable data, which fuels innovation.

  • BUILD MOMENTUM – spread the message within the sector, and contribute to supporting materials that anyone can use.
